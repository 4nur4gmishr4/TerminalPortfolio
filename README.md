# Welcome to your project

This is a portfolio website built with React, Vite, TypeScript, and Tailwind CSS.

## Project Overview

This project serves as a personal portfolio, showcasing skills, experience, and projects in an interactive terminal-like interface.

## Getting Started

### Prerequisites

Make sure you have Node.js (v18 or higher) and npm installed.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/your-portfolio.git
    cd your-portfolio/web
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Development Server

```bash
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser to see the application.

### Building for Production

```bash
npm run build
```

This will create a `dist` directory with the production-ready build.

## Customization

All personal data displayed in the terminal is sourced from `src/types/portfolio.ts`. You can edit this file to customize your portfolio content.

## Deployment

To deploy your project, you can use any static site hosting service (e.g., Vercel, Netlify, GitHub Pages). Simply deploy the `dist` folder generated after building the project.
